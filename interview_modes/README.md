# Interview Modes

> Choose the prep mode that matches your timeline, interview stage, and learning style.

## Available Modes

| Mode | Best For | Timeline | Focus |
|------|----------|----------|-------|
| [Quick Prep](quick_prep.md) | Last-minute revision | 3–7 days | High-signal questions only |
| [Deep Prep](deep_prep.md) | Comprehensive preparation | 30–60 days | Full progressive depth |
| [Interview Simulation](interview_simulation.md) | Mock interview practice | Per session | Round-based progression |
| [Debugging Mode](debugging_mode.md) | Incident/failure focus | 5–10 days | Level 4 questions across modules |
| [System Design Mode](system_design_mode.md) | Architecture preparation | 7–14 days | System and design questions |

## Mode Selection Guide

- **I have an interview in <7 days** → Quick Prep
- **I have 1–2 months to prepare** → Deep Prep
- **I want to practice mock rounds** → Interview Simulation
- **I'm senior and need debugging practice** → Debugging Mode
- **I'm preparing for architecture rounds** → System Design Mode
